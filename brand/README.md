# Le Nido — Brand Assets

Drop your brand files into the sub-folders below. The server and design files will reference these.

---

## /logos
- `logo-primary.svg` — Full logo (dark background)
- `logo-light.svg` — Full logo (light/white background)
- `logo-icon.svg` — Icon/mark only (favicon, app icon)
- `logo-primary.png` — PNG fallback (1000px wide minimum)
- `favicon.ico` — Browser favicon (32×32)

## /fonts
Drop any custom font files here (.woff2, .woff, .ttf).  
If using Google Fonts, just note the family name in `brand-guide.pdf` — no files needed.

## /colours
- `swatches.png` or `swatches.pdf` — Colour palette reference
- Current brand colours (already in CSS):
  - **Primary:** `#1c3d2e` (deep forest green)
  - **Accent:** `#c9a96e` (warm gold)
  - **Background:** `#f9f7f4` (warm off-white)

## Brand Guide File
Drop your brand guide PDF or Figma export directly in this `/brand` folder as `brand-guide.pdf`.

# Property Photo Assets

Drop photos for each property into the correct sub-folder below.

---

## Naming Convention

Use this format for all photos:

```
[property-id]-[room]-[number].jpg
```

**Examples:**
```
lekki-01-hero-01.jpg        ← main listing photo (used on cards + top of page)
lekki-01-hero-02.jpg        ← second hero
lekki-01-bedroom-01.jpg
lekki-01-bathroom-01.jpg
lekki-01-kitchen-01.jpg
lekki-01-living-01.jpg
lekki-01-exterior-01.jpg
```

---

## Specs

| Setting | Value |
|---|---|
| Format | JPG (preferred) or WebP |
| Resolution | Minimum 1200 × 800px |
| Orientation | Landscape (16:9 or 3:2) |
| File size | Under 500KB per photo (compress at https://squoosh.app) |
